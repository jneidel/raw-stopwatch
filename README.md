# raw-stopwatch

No-frills fork of [codealchemist/stopwatch-cli](https://github.com/codealchemist/stopwatch-cli).

![screenshot](screenshot.png)

## Install

```sh
npm install -g raw-stopwatch
```

## Usage

```sh
raw-stopwatch
```

**Keyboard interface:**

- `SPACE`: Pauses the timer.
- `ENTER`: Creates a lap.
- `q` or `ctrl+c`: Quit.
